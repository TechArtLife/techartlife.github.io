【系统配置】Windows中文版系统cmd命令行字体修改

目标​

解决中文版Windows操作系统中命令行终端默认字体是宋体的问题。

原因​

中文版命令行终端对程序员非常不友好。

解决方法​

1.原理​

修改 注册表 中的字体配置。

2.具体操作​

  1. 下载中英文通用的字体：https://github.com/yakumioto/YaHei-Consolas-Hybrid-1.12

  2. 修改注册表中相应位置代码页为936对应的字体。

    - （1） `Win+R` + `regedit`打开注册表编辑器。 
    - （2）切换路径到 `HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Console\TrueTypeFont`，双击名称为`936`对应的数据，修改为 `YaHei ConSolas Hybrid`。
