  • 【硬件】2022年小米Note3开启Root权限方法-官方工具

过程概览

  1. 解锁BL(Bootloader)，让手机进入Unlock状态。
  2. 刷入开发版MIUI系统，以启用Root权限管理功能。
  3. 在安全管家中开启Root权限功能，下载所需包，重启安装。
  4. 授予App Root权限。

详细步骤

1.BL解锁 - MIUI官方途径​

所需工具/软件​

  • miflash_unlock

    http://miuirom.xiaomi.com/rom/u1106245679/5.5.224.55/miflash_unlock-5.5.224.55.zip

Unlock解锁过程​

  1. 必须插入SIM卡。

  2. 在设备-我的设备-所有参数中连续点击MIUI版本信息，打开开发者模式。

  3. 进入设置-更多设置-开发者选项-设备解锁状态

    • 绑定小米账号和设备
    • 关闭WIFI，打开流量
    • 确认

  4. 使用MIUI官方工具解锁小米手机: http://www.miui.com/unlock/index.html

    • 下载解锁工具，解压。
    • 使用相同的小米账号登录解锁软件。
    • 在右上角设置中，点击检测驱动，安装所需驱动。

  5. 将手机关机，按住 音量- 和 开机键，等待进入Fastboot界面。

  6. 使用带有数据传输功能的USB数据线，将手机连接到电脑。

  7. 等待解锁工具识别到手机，点击 解锁，等待几秒，解锁完成后，手机会自动重启。

  8. 在设置-更多设置-开发者选项-设备解锁状态中可以看到手机是解锁状态。

2.刷入开发版MIUI - Fastboot 线刷​

所需工具/软件​

  • MiFlash:

    https://cdn.alsgp0.fds.api.mi-img.com/micomm/MiFlash2020-3-14-0.rar

  • MIUI 9.9.3 开发版:

    https://bigota.d.miui.com/9.9.3/jason_images_9.9.3_20190904.0000.00_9.0_cn_1701493577.tgz

刷机过程​

  1. 解压（两次）jasonimages_9.9.3_20190904.0000.00_9.0_cn_1701493577.tgz文件，最终解压出各种`flash*.bat`文件。

  2. 手机进入Fastboot模式，连接电脑。

  3. 打开MiFlash，先加载Android手机，然后配置右下角的刷机模式：将默认的全部删除并lock切换到全部删除，否则刷机完成又BL又锁上了。

  4. 点击刷机，等待刷机完成，手机重启。

  5. Ok，刷机结束。

3.开启Root权限​

  • 进入小米手机管家-权限管理-Root权限管理。
  • 根据操作开启Root功能，等待下载，重启。
  • 重启时会配置系统，需要等待较长时间，10min左右。
  • 开机后，再次进入小米手机管家-权限管理-Root权限管理中，可以看到已经有Root权限管理功能。

4.授予App Root权限​

  • 打开需要使用Root权限的应用，第一次请求使用Root权限时，会被系统直接拒绝，并弹出通知。
  • 在安全管家的Root权限管理中找到拒绝记录，对软件授予Root权限。
  • 重新打开App即可。
